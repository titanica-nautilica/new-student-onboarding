Calibration of radiocarbon ages

Interpolation

The files for the calibration curve usually have a different resolution to the internal storage resolution and so some form of interpolation is needed. This can either be linear or a cubic function depending on the setting in the system options (calib.xml: calib/linear). The cubic interpolation does not fit a spline function as this is very time consuming to calculate and can have some undesirable features such as large excursions between points. The cubic function used here gives a smooth curve with a continuous first differential but gives very little overall difference from the linear interpolation. Calendar and BP dates

Output and input can be given in terms of calendar years yCAL (AD/BC) or years before 1950 yBP (cal BP). The relationship between these is:
yCAL = 1950 - yBP (for AD)

yCAL = yBP - 1951 (for BC)
